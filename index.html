サンプルプログラムの概要 (app.xeh)

リポジトリには app.xeh というベンチマークスクリプトが含まれています。このスクリプトは、OSレベルのパイプストリーミングを使用し、単一のスキーマ下で異なるバックエンド環境を管理するという xeh 固有の価値を実証します。

データ管理ブロック

rootタグは、名前、データ型、および静的な値の厳格なマッピングを含む構造化されたJSON配列をインスタンス化します。エンジンはこの空間を main_space という識別子で分離します。

<root memory_name="main_space"> [{"name": "test1", "type": "string", "value": "test1aaaaa"}] </root>

マルチランタイム・オーケストレーション

subframeworkはリクエストをルーティングし、自動化されたPythonサブプロセスを実行します。エンジンは、低レイテンシ処理のために、Goのメモリ空間から構造化されたJSONデータをPythonの標準入力ストリーム（sys.stdin）にパイプラインで自動転送します。

for line in sys.stdin: packet = json.loads(line.strip()) print(f"Value of test1: {packet.get('data').get('test1')}")

リポジトリのファイル構造

以下は、xeh言語プロジェクトのリポジトリバンドル内に含まれる主要ファイルの役割一覧です。

ファイル / ディレクトリ 種別 機能説明
main.go Go ソース コアインタプリタエンジンです。XMLのコンパイル、メタデータの検証、サブプロセスの管理、およびリアルタイムなIPCストリームパイプ通信を処理します。
set.json JSON 設定 中心となるコマンド管理ハブです。独自のXMLタグを外部プラットフォームのバイナリ（Python、Rust、Java、Elixir、C++など）やプロジェクトのメタデータにマッピングします。
app.xeh xeh ソース 厳格な名前付きメモリ空間と複数言語のスクリプトブロックを含む、デフォルトの実行エントリポイントです。
index.html HTML 文書 コアアイデンティティのグラデーションバーを採用した、公式のユーザーポータル用資産です。開発者がリポジトリ資産をダウンロードするための案内を提供します。
LICENSE テキスト 文書 MITライセンスの条項に基づき、オープンソースとしての配布を法的に許諾する明示的なライセンスブロックです。
plugins/ ディレクトリ コンパイル中に生成される安全な出力スクリプトや、外部ランタイム定義を格納するために隔離されたワークスペースです。
